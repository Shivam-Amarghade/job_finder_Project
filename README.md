# job_finder_Project
JobPortal-Project


JobPortal Web Application

This is a full-stack Job Portal built using MERN stack, designed to simplify job searching and recruitment processes.

Key Features:

User roles: Job seekers & Recruiters

Job Posting: Recruiters can add, edit, and delete job listings

Job Application: Users can submit applications and upload resumes securely

Notifications: Recruiters get alerts for new applications

Frontend: Responsive React interface with smooth navigation

Backend: Node.js + Express API with MongoDB database for data storage

Security: Authentication & authorization implemented

Tech Stack:

Frontend: React.js, HTML, CSS, JavaScript

Backend: Node.js, Express.js

Database: MongoDB

Version Control: Git & GitHub

Purpose:

To provide a user-friendly and secure platform for job seekers and recruiters

To practice full-stack development skills with MERN stack
